# OpenDataAustria

### Back-End
##### SetUp
```sh
cd Back-End
npm install
npm start
```

### Prototyp
##### SetUp
- use Visual Studio Code
    - open folder with VSC
    - install extension Live Server
    - right click on index.html -> "Open with Live Server"
- use XAMPP
    - copy prototyp folder to htdocs
    - open http://localhost:8080/Prototyp/
